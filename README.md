# practice-synthetic-clinical-and-claims-data
1. This repo is for claims analytics and CDISC SDTM/ADaM demo project using synthetic data. 
2. This repository contains a beginner clinical data analytics portfolio project using synthetic healthcare claims-style data.

## Objective
To demonstrate skills in:
- data cleaning
- cohort creation
- descriptive analysis
- clinical/healthcare data documentation
- reproducible reporting in R

## Repository structure
- `data/` raw and processed data
- `R/` scripts
- `notebooks/` narrative analysis
- `outputs/` tables and figures
- `docs/` definitions and assumptions
